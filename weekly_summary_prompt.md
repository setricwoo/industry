# 周期行业周度跟踪 - 行业观点总结Prompt

## 使用说明
每周更新Excel数据后，将此prompt发送给Claude，Claude会根据当周数据和以下模板风格生成各行业观点总结。

---

## Prompt正文

请根据本周更新的Excel数据，按照以下各行业的表述风格，撰写本周的行业观点总结。

### 通用要求
1. **数据准确**：从Excel中提取最新的周度数据（价格、库存、开工率等），计算周环比变化
2. **简洁专业**：使用金融行业研报的表述风格，避免口语化
3. **逻辑清晰**：先陈述价格/数据变化，再分析原因，最后给出短期展望
4. **单位规范**：价格统一用"元/吨"或"美元/桶"，库存用"万吨"，开工率用"%"
5. **段落格式**：严格按照各行业的分段习惯，如有分点1）2）3），每个分点单独一行

## 一、煤炭

### 段落结构
通常为**一个自然段**，不分行。

### 开头格式
```
本周秦港现货煤环比[上升/下降]X%至XXX元/吨。
```

### 内容要点
- 秦港动力煤价格及环比变化
- 日耗情况（25省电厂日耗）
- 库存情况（秦皇岛港库存、25省电厂库存）
- 发电量数据（如有的话，注明旬度数据）
- 供需分析：淡旺季、补库情况、进口煤影响

### 历史案例

```
本周秦港现货煤环比下降2.1%至729元/吨。最新一期的旬度发电量（2月下旬）火电发电量同比下降11.2%，与春节错位有一定关系。淡季已至，电厂补库尚未开启，节后价格反弹告一段落。
```

```
本周秦港现货煤环比上升0.9%至745元/吨。节后库存持续下降，日耗从低位恢复，叠加海外天然气价格冲击与印尼煤炭生产配额影响，带动煤价有所反弹。
```

```
动力煤方面，本周秦港现货煤环比上升0.3%至694元/吨，终端日耗维持高位带动库存去化，国内价格小幅回升；但印度煤炭产量高增导致进口依赖度下降、LNG价格预期偏弱且印尼政策存不确定性的情况下，海运动力煤价格持续向上难度较大。焦煤方面，整体库存相对偏低，受印尼煤炭产量配额缩减影响价格出现了情绪层面脉冲式反弹；但蒙煤进口量持续处于高位，限制了价格上行空间。
```

```
本周秦港现货煤环比下降2.3%至685元/吨。从需求侧来看，今冬国内气温偏暖或通过取暖需求压制煤炭消费弹性；钢厂库存充足补库意愿不高，煤价偏弱。11月以来动力煤价格一路向下，与取暖季火电出力较弱不无关系，最新一期的旬度发电量（1月上旬）火电发电量同比下降0.3%，火力出电继续维持负增。
```

## 二、钢铁

### 段落结构
通常为**一个自然段**，不分行。

### 开头格式
```
价格方面，本周钢材价格有所[回升/下降]，螺纹XXXX元/吨，热卷XXXX元/吨。
```

### 内容要点
- 螺纹、热卷价格及变化
- 五大材表需及环比变化
- 五大材库存（社会+厂库）变化
- 高炉开工率/铁水产量
- 钢厂盈利情况
- 旺季/淡季判断

### 历史案例

```
价格方面，本周钢材价格有所回升，螺纹3250元/吨，热卷3280元/吨。旺季逐步开启，五大材表需提升15.4%，表观需求回升。五大材社会和厂库库存提升1.2%，库存提速放缓，但去库拐点尚未到来。当前黑色供需基本平衡，矛盾不大，价格震荡运行。
```

```
价格方面，本周钢材价格有所下降，螺纹3190元/吨，热卷3230元/吨。五大材表需提升4.4%，表观需求回升。五大材社会和厂库库存提升5.7%，库存继续环累，去库拐点尚未到来，旺季初期市场氛围一般，价格偏弱运行。
```

```
价格方面，本周钢材价格持平，螺纹3300元/吨，热卷3280元/吨。从基本面看，钢材供需两弱，五大材产量与表观需求均延续下降的趋势，钢材库存维持季节性去化。高炉端，高炉产出与盈利面见底。年末黑色系价格在商品市场偏强氛围中亦维持较强韧性，铁矿价格重新回到"反内卷"行情以来新高，钢材与螺纹价格亦有所反弹。在年关淡季黑色系或以博弈旺季预期为主，当前宏观预期偏暖，叠加基本面层面钢材库存维持去化，供需矛盾不算突出。
```

## 三、有色

### 段落结构
通常为**一个自然段**，不分行。

### 开头格式
```
价格方面，1#铜现货环比[上升/下降]X%至XXXXX元/吨，铝价[抬升/下降]X%为XXXXX元/吨，
```

### 内容要点
- 1#铜、A00铝价格及变化
- 铜社会库存、电解铝库存
- 铜精矿TC/RC
- 价格驱动因素（美元、地缘政治、需求等）

### 历史案例

```
价格方面，1#铜现货环比下降0.2%至100945元/吨，铝价抬升2.7%为25115元/吨，铜社会库存下降0.6%，为57.4万吨但交易所库存继续大幅增加。有色板块走势分化，铜受美元走强与需求预期冲击影响，价格环比回落。中东地缘冲突加剧，影响区域产能开工率，市场预期铝供应收紧，成为价格核心推手。
```

```
价格方面，1#铜现货环比下降1.0%至101165元/吨，铝价抬升4.4%为24445元/吨，本周铜社会库存上升1.3%，为137.1万吨，库存大幅累积。有色板块走势分化，铜受原油价格大涨带来的增长预期担忧影响，价格大幅下行。电解铝价格则受中东铝厂供应危机支撑。
```

```
价格方面，1#铜现货环比下降4.3%至100015元/吨，铝价下降6.1%为23145元/吨，价格高位波动，板块资金情绪退潮。本周铜社会库存上升1.3%，为137.1万吨，库存高位，伦铜转向contango。
```

## 四、石化

### 段落结构
分为**两个自然段**：
- 第一段：价格变动及原因
- 第二段：海外高频数据 + 国内高频数据

### 开头格式
```
价格方面，本周布伦特原油价格[上涨/下跌]X%至XX.X美元/桶，NYMEX HH天然气价格[上涨/下跌]X%至X.XX美元/百万英热。
```

### 内容要点
- 布伦特原油、NYMEX天然气价格及变化
- 价格变动原因（地缘政治、OPEC+、需求等）
- **海外高频数据**：美国炼厂开工率、商业原油库存、与5年同期对比
- **国内高频数据**：主营炼厂开工率、山东地炼开工负荷

### 历史案例

```

```

```

海外高频数据中，2月27日当周，美国炼厂开工率提升至89.2%，商业原油库存环比增加0.8%，绝对水平低于5年同期均值2.8%。国内高频数据中，截至3月4日，国内主营炼厂开工率82.2%，假期后汽油需求回落，主营下调原油加工量，考虑检修计划和原料风险，下周主营开工或继续下滑；山东地炼一次常减压开工负荷63.4%，较上周增加0.79ppt，在途原料或满足短期需求，预计短期内开工率受影响有限。
```

```
价格方面，本周布伦特原油价格下跌3.7%至68.05美元/桶，NYMEX HH天然气价格下跌23%至3.41美元/百万英热。美伊谈判压降原油溢价，北美气价回吐寒潮涨幅。

## 五、基础化工

### 段落结构
分为**多个自然段**：
1. 第一段：原油供应/化工品价格概述
2. 分点1）2）3）如有重大事件（每个分点单独一行）
3. PX-PTA-涤纶长丝产业链（单独成段）
   - 1）库存：...
   - 2）价格及盈利：...

### 内容要点
- 原油/炼化相关的宏观影响
- 重点化工品价格变动（如有重大变化）
- PX-PTA-涤纶长丝产业链（固定模块）

### 历史案例

**案例1 - 原油供应危机（多个分点+PX-PTA产业链）：**
```
原油供应紧张预期加强，炼厂预防性减产和成品油保供，炼油和衍生品利润大幅提升。由于霍尔木兹海峡的通行能力大幅下降，在过去的一周中，中东的原油供应减少了预计接近2000万桶/日。亚洲炼厂和中国主营炼厂进行了预防性减产，同时考虑中国的能源保供，预计可能对成品油出口进行限制。中国2025年汽柴煤出口总量3500万吨，占东南亚成品油销量的接近20%。新加坡炼油毛利大幅提升，321指数大幅提升至43美元/桶（环比上升30美元/桶）。

1）国内主营炼油利润有望提升，中国航煤的定价有望在下个月同步新加坡提升
2）民营炼化炼油毛利环比提升287%至721元/吨，px芳烃以及MTBE烷基化油等汽油调和料有望同步提升
3）东南亚炼油直接受益，特别有稳定油源或库存较为饱满的企业

此外，天然气价格的大幅上涨，推升欧洲能源成本价格，欧洲有产能的化工品比如维生素E、蛋氨酸、HDI等价格同样大幅上涨。

PX-PTA-涤纶长丝产业链：
1）库存：本周受成本方面提振，下游采购意愿提升，库存水平较上周下降。
2）价格及盈利：本周PTA价格上涨，长丝企业报价继续上调，本周POY/FDY/DTY价格分别上涨4.7%/4.5%/4.4%至7,410/7,630/8,530元/吨，长丝价差收窄9.9%/8.8%/4.5%至1,249/1,469/2,369元/吨。
```

**案例2 - 农药化工品（概述+PX-PTA产业链）：**
```
本周草铵膦价格上调，短期或仍有一定上行空间。据百川资讯，1月16日草铵膦市场价格4.6万元/吨，环比上周上涨0.21万元/吨，我们判断本次上涨的直接催化剂或在于草铵膦增值税出口退税将于4月1日起被取消，海外市场备货需求或已开始提前释放，向前看，考虑到当前价格下行业盈利微薄，我们认为抢单效应或有望在短期继续推升草铵膦价格，但考虑到长周期草铵膦供需关系仍维持宽松，草铵膦价格中枢大幅上行概率相对较低。1月16日己内酰胺开会讨论减产保价，己内酰胺价格或保持坚挺。

PX-PTA-涤纶长丝产业链：
1）库存：本周聚酯行业负荷仍处高位，对原料端存在一定刚性需求，且周内PTA产量小幅下滑，故目前PTA市场库存仍维持在相对低位。长丝方面，本周长丝市场局部产销表现尚可，各品种库存表现分化。
2）价格及盈利：长丝企业提前开启春节检修计划，同时油价上涨成本支撑尚可，本周POY/FDY/DTY价格分别上涨2.1%/1.3%/1.0%至6,690/6,890/7,825元/吨，长丝价差分别变化+16%/+8.5%/+4.1%至1,136/1,336/2,271元/吨。

**案例3 - 多品种涨价（多个化工品+PX-PTA产业链）：**

2）价格及盈利：本周PTA价格上涨（+3.3%至5,272元/吨）推动长丝成本上升，同时长丝企业修复现金流意愿偏强，长丝企业报价继续上调，本周POY/FDY/DTY价格分别上涨3.6%/4.1%/2.3%至6,965/7,220/8,060元/吨，长丝价差分别变化+3.4%/+5.8%/-1.2%至1,171/1,426/2,266元/吨。

## 注意事项

1. **数据来源**：所有数据来自Excel底稿，确保准确性
2. **环比计算**：(本周值-上周值)/上周值 × 100%
3. **特殊情况**：如遇节假日、重大事件，需在开头说明
4. **前瞻判断**：可适当加入短期展望，但避免过度预测
5. **一致性**：同一指标在不同行业的表述保持一致
6. **段落格式**：
   - 煤炭、钢铁、有色：一个自然段，不分行
   - 石化：分两段（价格 + 高频数据）
   - 基础化工：分多段，PX-PTA产业链单独成段，1）2）分点各占一行
